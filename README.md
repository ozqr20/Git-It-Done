# Git-It-Done
## Server Side API's
* Dynamically updated API using HTML attributes 
* Get familiar with API documentation 
* I used Fetch API to make requests, used DevTools to debug and check data info
* Handling errors while working with server-side API's
* Requesting data using HTTP to display data to the user
* Query parameters to pass info from one page to the other

* Live site https://ozqr20.github.io/Git-It-Done/
